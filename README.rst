Developer notes
===============

Please use a virtualenv to maintain this package, but I should not need to say that.

Grab the source from the SCM repository:

.. code:: console

  $ python setup.py develop
  $ pip install lastconf[dev]

Run the tests:

.. code:: console

  $ python setup.py test
  $ python run_tests.py
